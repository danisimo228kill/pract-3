# Ассемблер для УВМ (вариант 15)

Реализация первых двух этапов практической работы:
1. **Этап 1**: Перевод программы в промежуточное представление
2. **Этап 2**: Формирование машинного кода

## Спецификация УВМ (вариант 15)

### Команды:
| Мнемоника | Код | Размер | Описание |
|-----------|-----|--------|----------|
| `LOAD N`  | 13  | 3 байт | Загружает константу N на стек |
| `READ A`  | 14  | 5 байт | Читает значение из памяти по адресу A на стек |
| `WRITE`   | 0   | 1 байт | Записывает значение с вершины стека в память |
| `ROR`     | 7   | 1 байт | Побитовый циклический сдвиг вправо |

### Форматы команд:
1. **LOAD**: `AAAA BBBB BBBB BBBB B000`
   - A = 13 (4 бита)
   - B = константа (15 бит)

2. **READ**: `AAAA BBBB BBBB BBBB BBBB BBBB BBBB BBBB B000`
   - A = 14 (4 бита)
   - B = адрес (32 бита)

3. **WRITE**: `0000 0000`
   - A = 0 (4 бита)

4. **ROR**: `0111 0000`
   - A = 7 (4 бита)

### Тестовые последовательности из спецификации:
- `LOAD 988` → `0xCD 0x3D 0x00`
- `READ 468` → `0x4E 0x1D 0x00 0x00 0x00`
- `WRITE` → `0x00`
- `ROR` → `0x87`
